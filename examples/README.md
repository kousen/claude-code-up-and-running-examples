# Examples

Small, standalone, runnable code samples that illustrate a single point in the
book but are **not** part of any of the major projects (`lyrics-trainer/`,
`weather-app/`, `mockhub/`).

Organized by chapter, matching the numbering used in `prompts/` and
`transcripts/`.

| Chapter | Example | What it shows |
| --- | --- | --- |
| `ch03/` | `stateless_requests.py` | HTTP/API requests are stateless — the server keeps no memory between calls, which is why the Ch. 3 lyrics-trainer persists state client-side with `localStorage`. |
| `ch06/` | `commands/` + `skills/` | Custom-command-to-skill migration (`docs`), frontmatter tour skills (`modernize-java`, `onboard`, `security-review`), plus `image-prompt` and `osquery` demos from the chapter. |
