# Chapter 3 Plan Mode "what are alternatives?" prompt

Used for the "Ask Claude Code: What are alternatives?" sidebar in the Plan Mode
section. This is a standalone illustration, not the lyrics-trainer: it assumes
Claude Code has just produced a plan for a small database migration. Run it
against any concrete migration plan (a real small schema change works best).

```text
The plan you produced for the database migration uses ALTER TABLE ... USING. What other approaches exist, and when would each one be right?
```
