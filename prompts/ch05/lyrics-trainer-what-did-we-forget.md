# ch05 lyrics trainer "What Did We Forget?" prompts

Ken's standard wrap-up ritual for any project, run as two separate headless
prompts against the ch05-navigation snapshot.

Prompt 1 (step-back review):

```text
Look at the overall system again. What do you think? Are there changes we
should make? Any cleanup steps we should do?
```

Prompt 2 (feature discovery):

```text
What have we forgotten to include that we should consider adding? What do
other similar apps have that we don't? What would make our app unique?
```
