# Chapter 5 post-work summary prompt

The "what did you actually do, and which boundaries did you cross?" wrap-up
(ch05 ~lines 691-701). Pair the captured summary with a real `git diff` /
`pytest` so the chapter can make the point that the agent's summary is useful
but the repository is the source of truth.

```text
Summarize:

- files changed
- tests added or updated
- commands run
- whether any network, secret, dependency, git publication, or deployment boundary was crossed
- anything you wanted to do but did not have permission to do
```
